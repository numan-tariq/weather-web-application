## Functionalities

- Weather forecast
    - Temperature.
    - Weather Descritpion.

- Display weather with the help of City Name
    - Search city via input field.
    - Click 'Show Weather'

- Show Temperature in
    - Celsius
    - Fahrenheit
    - Kelvin
